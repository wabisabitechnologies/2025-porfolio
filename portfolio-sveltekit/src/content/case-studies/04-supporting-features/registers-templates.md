# Registers Templates: Pre-Built Compliance Framework

## Overview

Statutory registers are complex:
- Register of Directors (columns: Name, Appointment Date, Address, etc.)
- Register of Members (columns: Name, Address, Shareholding, etc.)
- Register of Beneficial Owners (columns: Name, ID, Ownership %, Verification Status, etc.)
- Share Ledger (columns: Date, From, To, Quantity, Consideration, etc.)

**Traditional Problem:**
- Companies don't know what a register should include
- They create custom registers (often incomplete, not compliant)
- Auditors find missing columns or incorrect format
- Year later: Try to add missing information (too late, record incomplete)

---

## How SFL Solves It

**Pre-Built Templates:**
- Register template for each register type
- Includes: All CBK/CMA required columns
- Explained: What each column means, why it's important
- Customizable: Add extra columns if needed

**Template Benefits:**
- Compliance assured: Includes all regulatory requirements
- Complete: No missing columns or fields
- Professional: Auditor sees properly formatted register
- Flexible: Company can customize if needed

**Example - Register of Beneficial Owners Template:**

| Column | Purpose | Required? | Format |
|--------|---------|-----------|--------|
| Name | BO's full name | Yes | Text |
| ID Type | Kenyan Citizen, Foreign, Company, etc. | Yes | Dropdown |
| ID Number | Passport/National ID/Company Reg | Yes | Text |
| Ownership % | Share percentage owned | Yes | Number |
| Voting % | Voting power percentage | Yes | Number |
| Verification Status | Verified/Pending/Unverified | Yes | Status |
| Date Verified | When was BO verified? | Yes | Date |
| Joint Ownership | If joint with others, who? | No | Text (if applicable) |
